منصة نادي اللغة الإنجليزية عبر الإنترنت — Online English Club Platform
منصّة لممارسة التحدّث بالإنجليزية مع مدرّس حقيقي بانتظام وفي الأوقات التي تناسب المتعلّم، مع الذكاء الاصطناعي في دور مساعد لا يحلّ محلّ المدرّس. الاسم هو الوعد: نادٍ تنتمي إليه، لا دورة تُنهيها.

هذه وثيقة تشرح فكرة النظام بالتفصيل: الرؤية والأهداف، ثم رحلة العمل الكاملة من الطالب إلى الإدارة إلى الأستاذ، ثم المفاهيم الأساسية ودور الذكاء الاصطناعي.

١. الرؤية والأهداف

  • التحدّث أولًا: ما ينقص أغلب المتعلّمين ليس القواعد، بل وقت التحدّث مع إنسان. المنصّة تجعل هذا الوقت سهل الحصول وسهل الاستمرار.
  • الإنسان في المركز والذكاء الاصطناعي على الأطراف: كل جلسة محادثة حقيقية مع مدرّس مُدقَّق؛ الذكاء الاصطناعي لا يُدرّس ولا يحلّ محلّ المدرّس، بل يساعد فقط (تحديد المستوى، اقتراح الأسئلة، تحليل ما بعد الجلسة، تدريب فردي اختياري).
  • الأوقات أولًا لا القوائم: المتعلّم يقول متى هو متاح، والنظام يتكفّل بالمطابقة — دون البحث في قوائم مواضيع أو أساتذة.
  • تقدّم يُرى: كل جلسة تنتهي بتقرير ذكاء اصطناعي، ويُتابَع التقدّم مهارةً مهارة عبر الزمن.
  • متاح حيث يصعب الدفع بالبطاقة: الدفع عبر تحويل بنكي محلي تعتمده الإدارة — دون بطاقة.
  • ثنائية اللغة افتراضيًا: الواجهة كاملة بالعربية والإنجليزية (مع الاتجاه من اليمين لليسار)، فلا تكون المنصّة نفسها عائقًا.

٢. الفكرة الأساسية — حلقة الأدوار الثلاثة

قلب المنصّة حلقة من ثلاثة أدوار: الطالب يعرض أوقاته، والإدارة تُسنِد وتعتمد، والأستاذ يحضّر ويُدرّس، والجميع يرى النتيجة.

١
الطالب — الأوقات المتاحة: يحدّد في صفحته أيام وساعات الأسبوع التي يستطيع الحضور فيها. لا يختار موضوعًا — الدرس مهمّة الأستاذ.
٢
النظام — الإسناد التلقائي: لكل وقت يختار النظام أقرب أستاذ متاح (أستاذ يغطّي وقته ذلك، مع توزيع عادل للطلاب).
٣
الإدارة — بوابة المراجعة: ترى أوقات كل طالب مع المدرّس المُسنَد وقائمة بقية الأساتذة المتاحين في ذلك الوقت؛ تؤكّد أو تغيّر المدرّس أو ترفض الوقت بملاحظة، ثم تعتمد.
٤
توليد الحجوزات: عند الاعتماد تُنشأ الجلسات للأسابيع القادمة، وتُخصم جلسة مقابل رصيد واحد، ويُشعَر الأستاذ. ومهمّة خلفية تُجدّد الأسابيع القادمة تلقائيًا.
٥
الأستاذ — تحضير الدرس: يكتب لكل جلسة عنوانًا وأسئلة نقاش (نصًّا حرًّا، مع زر اقتراح بالذكاء الاصطناعي)، ويُسمح بالتحضير خلال ٣ أيام قبل الجلسة فقط ليبقى الدرس محدّثًا.
٦
الطالب — الكشف قبل ساعة: يظهر عنوان الدرس وأسئلته للطالب قبل الجلسة بساعة ليجهّز نفسه — لا قبل ذلك.
٧
الجلسة المباشرة: غرفة فيديو فيها سبورة ومحادثة ومشاركة شاشة وملفات وتسجيل.
٨
التقرير والتقدّم: بعد الجلسة تقرير ذكاء اصطناعي يقيّم القواعد والمفردات والطلاقة والنطق والثقة، مع نقاط القوة والأخطاء والواجب، ويُتابَع التقدّم عبر الزمن.

٣. الرحلات بالتفصيل

👩‍🎓 الطالب

يسجّل ← يختار هدفًا ← اختبار تحديد المستوى (كتابي + شفهي) ← يدفع بتحويل بنكي وينتظر اعتماد الإدارة ← يحدّد أوقاته المتاحة ← تُحجز جلساته بعد المراجعة ← يرى كل درس قبل ساعة ← يحضر الجلسة المباشرة ← يقرأ التقرير ويتابع تقدّمه. واختياريًا يتدرّب منفردًا مع المعلّم الذكي.

🛡️ الإدارة

تعتمد/ترفض إثباتات الدفع (فتُفعَّل الاشتراكات والأرصدة) ← تعالج طابور طلبات الجدولة (تؤكّد/تغيّر المدرّس، تعتمد أو ترفض أوقات كل طالب) ← تدير الأعضاء والأساتذة والخطط والحجوزات ← تتابع مؤشّرات الأعمال وسجلّ التدقيق. كل إجراء مهمّ مُسجَّل.

👨‍🏫 الأستاذ

يضبط أوقاته المتاحة أسبوعيًا ← يُسنَد إليه الطلاب الذين تناسب أوقاتهم ← في تحضير الدرس يكتب لكل جلسة عنوانًا وأسئلة (بمساعدة الذكاء الاصطناعي) خلال نافذة الأيام الثلاثة ← يُدرّس الجلسة المباشرة ← يراجع/يعتمد التقرير. وله ملف عام (سيرة).

٤. المفاهيم الأساسية

  • الجدولة القائمة على الأوقات: الطلاب يختارون الأوقات، والمطابقة بالأساتذة تتم بالتوفّر لا بالاختيار من قائمة.
  • بوابة مراجعة الإدارة: لا تُنشأ أي جلسة قبل اعتماد الإدارة لأوقات الطالب والمدرّس المُسنَد. أي تعديل يعيد الاختيار إلى «بانتظار المراجعة».
  • الأرصدة والاشتراكات: خطط حسب المدة، تُفعَّل بعد التحويل البنكي واعتماد الإدارة. كل جلسة تستهلك رصيدًا، وشحن الرصيد يُفعّل فورًا أي جدول معتمد كان ينقصه رصيد.
  • دروس لكل جلسة: الدرس (عنوان + أسئلة) على الجلسة نفسها، يكتبه الأستاذ خلال ٣ أيام قبلها، ويظهر للطالب قبل ساعة — منفصل تمامًا عن أي مكتبة مواضيع قديمة.
  • احترام الإلغاء: إن ألغى الطالب جلسة متكرّرة، لا يعيد المُولّد إنشاءها أو خصم رصيدها.
  • المعلّم الذكي (اختياري): اشتراك منفصل لتدريب فردي قصير (٥ دقائق) بصوت ودرجة قابلين للضبط.
  • ثنائية اللغة: واجهة كاملة عربي/إنجليزي مع دعم الاتجاه من اليمين لليسار، والتواريخ تُعرَض بلغة الواجهة.

٥. ماذا يفعل الذكاء الاصطناعي — وماذا لا يفعل

الذكاء الاصطناعي أداة مساعدة لا معلّم:

  • يقيّم اختبار تحديد المستوى.
  • يقترح أسئلة النقاش من عنوان الدرس (يعدّلها الأستاذ ويعتمدها).
  • يحلّل الجلسة بعدها إلى تقرير منظّم.
  • يشغّل المعلّم الذكي للتدريب الفردي القصير.

وعند غياب مفاتيح مزوّدي الخدمة يعمل ببديل آمن ثابت، فلا يتوقّف النظام. المحادثة المباشرة بشرية دائمًا.
